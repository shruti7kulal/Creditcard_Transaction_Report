# Credit Card Transaction Analysis Dashboard

## Project Overview

### Objective

To develop a comprehensive credit card weekly dashboard that provides real-time insights into key performance metrics and trends, enabling stakeholders to monitor and analyze credit card operations effectively.

This project presents an end-to-end Credit Card Transaction Analysis Dashboard built using PostgreSQL and Power BI. The solution transforms raw transaction and customer data into actionable business insights through interactive visualizations and KPI-driven reporting.

The dashboard helps stakeholders monitor revenue performance, customer behavior, transaction trends, and operational metrics to support data-driven decision-making.

---

## Business Problem

Financial institutions generate large volumes of credit card transaction data every week. Analyzing this data manually is time-consuming and often fails to provide actionable insights.

This dashboard was designed to:

- Monitor weekly revenue performance
- Track transaction trends and customer activity
- Analyze customer demographics and spending behavior
- Identify high-performing customer segments
- Measure customer satisfaction and card usage patterns
- Support business decision-making through KPI-driven reporting

## Key Insights

- Revenue increased by 28.8% Week-over-Week during Week 53.
- Total revenue reached 57M.
- Total transaction amount exceeded 46M.
- Male customers contributing more revenue 31M,female 26M
- Blue & Silver credit card are contributing to 93% of overall transactions
- Texas, New York, and California generated nearly 68% of total revenue.
- Customer activation rate remained strong at 57.5%.
- Delinquent accounts represented 6.06% of the customer base.
